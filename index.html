About me

Hi! I'm Melody! I am about to graduate from iO Academy as a full stack software developer. In a previous life, I worked as an astrophysicist in France and in the UK for 9 years, and then as a operational researcher in international trade for UK's civil service.

I'm curious, I love a good challenge - especially if it gets me outside of my comfort zone, and I love coding. I am looking forward to pursue these interests in my new career as a software developer.

Outside work, I enjoy singing with my a cappella choir, playing piano, sewing, crochet and all things space-related!

Pro-To-Type

This is a typing speed game where players have 60s to copy a randomly generated text. At the end of the game they can see their typing speed and the accuracy scores. We used Javascript, HTML and CSS to create this app. I worked on various functionalities of this app such as fetching a random text to copy from an API, comparing the player's input to the reference text and a timer. I was also scrum master for this project.

Github Live

Food Delivery App

We developed the frontend for a food delivery app using Bootstrap CSS and React Javascript. I worked on the feature that allows diners to choose a restaurant, fetches its menu from a supplied API and displays it on the webpage.

Github

Project management API

We created a backend API for a project management app with PHP and SQL. We used OOP to create endpoints following the SOLID principles for the application architecture, allowing users to display all the projects from the database or a specific project. I worked on the class containing the details of a project, unit testing and error handling.

Github

iO Academy Portal

We contributed to the development of a management software for the iO Academy, which would handle the logistics of its training courses (i.e. creating student profiles, registering them in a course etc.). We worked on the implementation of a feature that would allow a student to edit their profile to add additional information. I worked on the PHP backend. I implemented a new endpoint for the API so that the new data are validated, sanitised and added in the mySQL database.
