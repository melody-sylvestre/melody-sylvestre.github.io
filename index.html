About me

Portfolio

As a student of the iO Academy full stack software development course, I have worked on several individual and team projects. You will find below a brief description for each of them.

Pro-To-Type

This is a typing speed game in JavaScript where players have 60s to copy a randomly generated text. At the end of the game they can see their typing speed and the accuracy scores. I worked on various functionalities of this app such as fetching a random text to copy from an API, comparing the player's input to the reference text and a timer. I was also scrum master for this project.

Food Delivery App

We developed the frontend for a food delivery app using Bootstrap CSS and React Javascript. I worked on the feature that allows diners to choose a restaurant, fetches its menu from a supplied API and displays it on the webpage.

Project management API

We created a backend API for a project management app with PHP and SQL. We used OOP to create endpoints following the SOLID principles for the application architecture, allowing users to display all the projects from the database or a specific project. I worked on the class containing the details of a project, unit testing and error handling.
